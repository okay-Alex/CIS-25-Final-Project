<h1>Task Tracker</h1>
<p>Simple command-line task tracker, allowing tasks to be assigned tags and dates. Also allows tasks to be filtered and sorted by tags and dates.</p>
<h2> Commands </h2>
<ol>
  <li>Look at current tasks with applied filters</li>
  <li>Filter by tag</li>
  <li>Sort by date</li>
  <li>Create task</li>
  <li>Archive task</li>
  <li>Show archived</li>
  <li>Exit</li>
</ol>
